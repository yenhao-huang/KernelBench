depthwise convolution kernel.

A depthwise convolution is a convolution where each input channel is convolved with its own unique kernel.
The input shape is (N, C, H, W).
The weight shape is (C, 1, K, K) or (C, C, K, K) with groups=C.
The output shape is (N, C, H_out, W_out).

I will implement a kernel that maps each thread to one output element (N, C, H_out, W_out).
The kernel will iterate over the KxK window.
